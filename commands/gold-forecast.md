# /gold-forecast 命令

获取黄金行情多维分析仪表盘，包含技术面、基本面、博弈面、机构持仓综合评分，支持多周期趋势分析和操作建议。

## 用法

```
/gold-forecast
```

## 分析维度

- **技术面 (25%)**：价格趋势、支撑阻力、动量指标
- **基本面 (30%)**：美元指数、美联储政策、通胀、地缘政治
- **博弈面 (25%)**：ETF持仓、央行购金、多空比例
- **持仓面 (20%)**：CFTC COT 报告、Top 10 机构持仓变化

## 多周期分析

- **1周**：短期动量、5日均线
- **1月**：短中期趋势、20日均线
- **3月**：中期趋势、季度高低点
- **6月**：中长期趋势判断
- **1年**：长期趋势、年度走势

## 数据来源

- 国际金价：Metals.live API
- 国内金价：新浪财经
- 黄金 ETF：东方财富
- 市场要闻：金十数据
- CFTC 持仓：金十数据中心
- 美联储政策：CME FedWatch
- 经济日历：金十数据

## 输出内容

```
/gold-forecast

→ 输出黄金行情分析仪表盘，包含：
  - 四大品种实时行情（伦敦金、上海金、黄金ETF、美元指数）
  - 多周期趋势分析（1周/1月/3月/6月/1年）
  - Top 10 机构持仓分析（CFTC COT 报告）
  - 技术面/基本面/博弈面/持仓面四维评分
  - 美联储政策追踪与利率预期
  - 本周重要事件预警
  - 综合操作建议（短期/中期/长期/风险提示）
  - 近期关键财经要闻
  - 可选保存分析结果到知识库
```
