# Gold Forecast Skill 设计文档

## 概述

黄金行情预测 Skill，为个人黄金投资提供决策参考。整合技术面、基本面、博弈面三维分析，输出结构化仪表盘和操作建议。

## 核心决策

| 项目 | 决策 |
|------|------|
| **定位** | 个人黄金投资决策参考工具 |
| **数据源** | 免费 API + 财经网站抓取 |
| **分析维度** | 技术面 (30%) + 基本面 (40%) + 博弈面 (30%) |
| **输出形式** | 结构化仪表盘 + 分期限操作建议 |
| **调用方式** | `/gold-forecast` 手动触发 |
| **覆盖品种** | 国际金 + 国内金 + 黄金 ETF |

## 数据获取层

### 数据源配置

| 数据类别 | 数据源 | 获取方式 | 更新频率 |
|---------|--------|---------|---------|
| **国际金价** | Gold API / Metals.live | WebFetch API | 实时 |
| **美元指数** | 新浪财经 | Playwright 抓取 | 实时 |
| **国内金价** | 上海黄金交易所/新浪 | Playwright 抓取 | 交易时段 |
| **黄金 ETF** | 东方财富 | Playwright 抓取 | 交易时段 |
| **ETF 持仓** | SPDR Gold Trust | WebFetch | 每日 |
| **CFTC 持仓** | 金十数据/财经网站 | Playwright 抓取 | 每周五 |
| **央行购金** | 世界黄金协会 | WebFetch | 每月 |
| **财经要闻** | 金十数据/华尔街见闻 | Playwright 抓取 | 实时 |

### 数据获取策略

1. **并行获取**：价格数据 + 持仓数据 + 要闻 同时抓取
2. **容错处理**：单一数据源失败不影响整体分析
3. **缓存逻辑**：低频数据（央行购金、CFTC）可缓存复用

## 分析引擎

### 技术面分析 (30%)

| 指标 | 计算方式 | 信号判断 |
|------|---------|---------|
| **趋势方向** | 对比 5日/20日/60日均价 | 多头/空头/震荡排列 |
| **支撑阻力** | 近期高低点 + 整数关口 | 距离当前价的百分比 |
| **动量指标** | 近N日涨跌幅 | 强势/弱势/超买/超卖 |
| **波动率** | 近期振幅统计 | 高波动/低波动 |

### 基本面分析 (40%)

| 因素 | 关注点 | 影响逻辑 |
|------|--------|---------|
| **美元指数** | 当前点位 + 趋势 | 美元强→金价承压 |
| **美联储政策** | 利率预期 + 近期表态 | 降息预期→利好黄金 |
| **通胀数据** | CPI/PCE 趋势 | 高通胀→避险需求 |
| **地缘政治** | 热点事件 | 风险事件→避险买盘 |

### 博弈面分析 (30%)

| 指标 | 信号含义 |
|------|---------|
| **ETF 持仓增减** | 增持→机构看多；减持→机构离场 |
| **CFTC 净多头** | 极端高位→警惕回调；极端低位→可能反弹 |
| **央行购金** | 持续增持→长期支撑 |
| **多空比例** | 散户极端看多→反向指标 |

### 综合评分算法

```
技术面评分 (30%) + 基本面评分 (40%) + 博弈面评分 (30%) = 综合评分

评分区间：
  80-100: 强烈看多
  60-79:  谨慎看多
  40-59:  中性观望
  20-39:  谨慎看空
  0-19:   强烈看空
```
