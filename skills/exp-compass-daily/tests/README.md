## 测试运行步骤(由 AI 在真实流程中执行)

```
Step 1  Read fixtures/baseline-2026-05-07.json
Step 2  按 SKILL.md 模板 + 6 条撰写约束生成 MD
        (此步可写到 /tmp/exp-compass-test-output.md,不入知识库)
Step 3  对照本 README "Ground Truth" 节跑 C1-C6 自检
Step 4  6 项全过 → P3 验收通过
        任一失败 → 排查 SKILL.md 约束是否有歧义或漏点
```
