# Gold Forecast - 黄金行情预测

## Overview

多维度黄金行情分析工具，整合技术面、基本面、博弈面、机构持仓数据，支持多周期分析，输出结构化仪表盘和投资建议，并可保存至知识库。

**必须显示提示：** 使用此 Skill 时，必须在开始时输出：
> 🥇 **正在获取黄金行情数据...**
> 数据来源：国际金价 API、新浪财经、东方财富、金十数据、CFTC、CME FedWatch

## When to Use

**自动触发场景：**
- 用户询问黄金价格、走势
- 用户询问是否应该买入/卖出黄金
- 用户询问黄金 ETF 投资
- 用户询问避险资产配置
- 用户询问大户持仓、机构动向
- 用户询问美联储政策对黄金影响
- 用户询问本周/近期重要事件

### Phase 2: 多维分析

#### 2.1 技术面分析 (权重 30%)

| 指标 | 计算方法 | 评分规则 |
|------|---------|---------|
| 趋势方向 | 当前价 vs 5日/20日均价 | 多头排列 +30，空头 -30，震荡 0 |
| 支撑阻力 | 整数关口±1% | 接近支撑 +20，接近阻力 -20 |
| 动量 | 近 5 日涨跌幅 | >3% +25, 1-3% +15, -1~1% 0, <-3% -25 |
| 波动率 | 近 5 日振幅 | 低波动 +10，高波动 -10 |

技术面评分 = 各指标得分汇总，归一化到 0-100

#### 2.2 基本面分析 (权重 40%)

| 因素 | 评判标准 | 评分规则 |
|------|---------|---------|
| 美元指数 | DXY 点位和趋势 | <100 +25, 100-103 0, >103 -25 |
| 美联储政策 | 降息/加息预期 | 降息预期 +30，加息预期 -30 |
| 通胀数据 | CPI 趋势 | 高通胀 +20，低通胀 -10 |
| 地缘政治 | 风险事件 | 有重大事件 +25 |

基本面评分 = 各因素得分汇总，归一化到 0-100

#### 2.3 博弈面分析 (权重 30%)

| 指标 | 数据来源 | 评分规则 |
|------|---------|---------|
| ETF 持仓 | SPDR GLD | 周增持 +25，周减持 -25 |
| CFTC 净多头 | 金十数据 | 极端高位 -20，极端低位 +20 |
| 央行购金 | 世界黄金协会 | 持续增持 +15 |
| 多空比例 | 期货市场 | 散户极端看多 -15（反向） |

博弈面评分 = 各指标得分汇总，归一化到 0-100

#### 2.4 Top 10 持仓分析 (新增模块)

**CFTC COT 报告解读：**

| 持仓类型 | 解读方法 | 信号判断 |
|---------|---------|---------|
| 大型投机者净多头 | 周变化量 | 增加 → 看多信号；减少 → 看空信号 |
| 商业套保者净空头 | 对冲行为 | 增加 → 现货商看涨预期 |
| 持仓集中度 | Top 4/8 占比 | 高集中度 → 趋势可能延续 |
| 极端值预警 | 历史分位数 | >90% 或 <10% → 反转预警 |

**持仓信号综合判断：**
```
强看多信号 🟢🟢:
  - 大型投机者净多头周增 > 10%
  - 商业套保者空头减少
  - 持仓总量增加

强看空信号 🔴🔴:
  - 大型投机者净多头周减 > 10%
  - 商业套保者空头增加
  - 持仓总量下降

背离预警 ⚠️:
  - 价格新高但净多头下降 → 顶部预警
  - 价格新低但净多头增加 → 底部预警
```

#### 2.5 多周期趋势分析 (新增模块)

**各周期趋势评估表：**

| 周期 | 趋势判断标准 | 权重 |
|------|-------------|------|
| 1周 | 短期动量，价格 vs 5日均线 | 15% |
| 1月 | 短中期趋势，20日均线方向 | 25% |
| 3月 | 中期趋势，季度高低点位置 | 25% |
| 6月 | 中长期趋势，半年均线 | 20% |
| 1年 | 长期趋势，年度方向 | 15% |

**趋势共振分析：**
```
强趋势共振 (同向):
  - 所有周期趋势一致 → 趋势延续概率高
  - 短周期领先转向 → 变盘预警

趋势分化 (异向):
  - 短期空中期多 → 回调买入机会
  - 短期多中期空 → 反弹卖出机会
```

#### 2.7 综合评分

```
综合评分 = 技术面 × 0.25 + 基本面 × 0.30 + 博弈面 × 0.25 + 持仓面 × 0.20

评分区间：
  80-100: 强烈看多 🟢🟢
  60-79:  谨慎看多 🟢
  40-59:  中性观望 🟡
  20-39:  谨慎看空 🔴
  0-19:   强烈看空 🔴🔴

多周期共振加成：
  - 所有周期同向 → 评分 ×1.1（上限100）
  - 短中期分化 → 评分 ×0.9
```

## 容错处理

1. **单数据源失败**：标注"数据获取失败"，不影响其他分析
2. **API 限流**：使用备用数据源或缓存数据
3. **网页结构变化**：提示用户数据可能不准确
4. **Playwright 抓取失败**：记录失败原因，尝试备用 URL
5. **CFTC 数据延迟**：COT 报告每周五发布，注明数据日期

## Common Mistakes

| 错误 | 正确 |
|------|------|
| 只看单一指标 | 综合多维分析（技术+基本+博弈+持仓） |
| 忽略数据时效 | 标注数据获取时间，CFTC 数据注明发布日期 |
| 给出绝对建议 | 强调仅供参考，投资需谨慎 |
| 忽略风险提示 | 每次都包含风险因素 |
| 忽略多周期 | 分析各周期趋势共振情况 |
| 忽略事件日历 | 提醒本周重要事件及影响 |
| 不保存分析 | 询问是否保存到知识库 |
| 单一数据源 | 多数据源交叉验证 |
